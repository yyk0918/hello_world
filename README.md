# hello_world

이 폴더에는 네이버 스타일의 간단한 홈페이지 예제가 들어 있습니다.

파일
- index.html: 메인 HTML 파일 (홈페이지)
- styles.css: 스타일시트

로컬에서 미리보기
1) 브라우저에서 index.html을 직접 열기
2) 또는 파이썬 간단 서버 사용 (권장)

```powershell
cd hello_world
python -m http.server 8000
# 브라우저에서 http://localhost:8000 열기
```

원하시면 이 레이아웃을 더 네이버와 비슷하게 다듬거나, 반응형/모바일 뷰를 개선해드릴게요.
